# 00 Required Sources

## Purpose

Define the minimum source collection process required before any company analysis begins.

This document is deterministic, operational, and non-analytical. It exists to prevent investment conclusions from being produced before sufficient primary source material has been reviewed.

## Mandatory Sources

For any public company, collect and review the following before analysis begins:

- Latest annual report: 10-K, 20-F, annual report, or local equivalent.
- Latest quarterly report or earnings release.
- Latest earnings call transcript.
- Latest investor presentation, if available.
- Segment disclosures and relevant footnotes.
- Current market capitalization.
- Current share count.
- Current cash and cash equivalents.
- Current debt data.

If any mandatory source is unavailable, mark it unavailable in the source log and document the limitation.

## Recommended Sources

Collect recommended sources when they are relevant and available:

- Competitor filings.
- Industry reports.
- Regulatory filings.
- Shareholder letters.
- Investor day presentations.
- Credit rating reports.
- Proxy statements.
- Company press releases.
- Exchange filings or local market disclosures.

Recommended sources do not replace mandatory sources.

## Source Log Requirements

Create a source log before analysis begins. The source log must include one row per source.

Required fields:

- Source name.
- Date.
- Source type.
- Status: reviewed or unavailable.
- Notes.

Use this format:

| Source Name | Date | Source Type | Status | Notes |
| --- | --- | --- | --- | --- |
|  |  |  | Reviewed / Unavailable |  |

## Missing Information Rules

If required information is missing:

- Explicitly state each missing mandatory source or data item.
- Mark each missing item as unavailable in the source log.
- Explain how the missing information limits confidence.
- Do not replace missing primary sources with news summaries, blog posts, social media, or market commentary.
- Do not produce investment conclusions that depend on unavailable information.

Secondary sources may provide context, but they do not satisfy mandatory source requirements unless they reproduce the original primary source data clearly and verifiably.

## Completion Criteria

Research intake is complete only when all of the following are true:

- All mandatory sources have been reviewed or explicitly marked unavailable.
- Current market capitalization, share count, cash, and debt data have been reviewed or explicitly marked unavailable.
- A source log exists.
- Missing information is documented.
- Decision limitations from missing information are documented.

If any completion criterion is not met, stop and complete the source collection process before beginning business analysis, valuation work, thesis development, or recommendation drafting.
